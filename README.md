# js-cnpj-validation

Lib for CNPJ validation

```js
const cnpj = require('js-cnpj-validation');  
console.log(cpf.isValidCNPJ("49.623.477/0001-12"))

console.log(isValidCNPJ("49.623.477/0001-12"))
```

Output:
true
